## Getting Started

1. **Fork the Repository**: Click the "Fork" button at the top of this page.
2. **Clone Your Fork**:

   ```bash
   git clone https://github.com/your-username/agent-semantic-protocol.git
   ```

3. **Install Dependencies**:

   ```bash
   go mod download
   ```

4. **Run Tests**: Ensure everything is working before making changes.

   ```bash
   make test
   ```

---

## Contribution Guidelines

- Follow [Conventional Commits](https://www.conventionalcommits.org/):
  - `feat:` for new features
  - `fix:` for bug fixes
  - `docs:` for documentation updates
- Write clear, concise commit messages.
- Ensure all tests pass before opening a pull request.
- Open a descriptive pull request with a clear title and summary of changes.

---

## Branch Naming Conventions

- **`main`**: The main branch.
- **`develop`**: The development branch.
- **`feature/*`**: Feature branches.
- **`bugfix/*`**: Bug fix branches.
- **`hotfix/*`**: Hotfix branches.

---

## Pull Request Criteria

- **Code Quality**: All code must pass linting and static analysis.
- **Tests**: All tests must pass.
- **Documentation**: All changes must be documented.
- **Review**: All PRs must be reviewed by at least one other contributor.
